# Conecta

> Mensageiro web moderno inspirado na simplicidade dos comunicadores clássicos.

## Sobre o Projeto

O Conecta é um Progressive Web App (PWA) desenvolvido para demonstrar conhecimentos em desenvolvimento web moderno utilizando Firebase.

O sistema permite comunicação privada entre usuários autenticados através de mensagens em tempo real, utilizando uma arquitetura totalmente Serverless.

Este projeto faz parte do portfólio de desenvolvimento de Marcelo José de Paula.

---

## Objetivos

- Demonstrar conhecimentos em desenvolvimento web.
- Aplicar arquitetura Serverless.
- Utilizar os serviços gratuitos do Firebase.
- Desenvolver uma aplicação moderna para comunicação em tempo real.
- Criar um projeto completo para portfólio profissional.
